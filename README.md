# Spin to Speak
### AI-Supported Impromptu Speaking Practice

A browser-based spinner that delivers random prompts and short prep times to help students build confidence, structure, and composure in public speaking. Designed for use in Public Speaking courses — in person or online.

Built for multilingual, first-generation, and apprehensive speakers. Works as a warm-up, an asynchronous tool, or a structured unit introducing Joshua Gunn's Rule of 3 (introduction, body, conclusion).

## How to Use

1. Download `spinner.html`
2. Open it in any browser — no installation, no login, no internet required
3. Click the spinner to receive a random prompt
4. Set a prep time (30 seconds, 1 minute, or 2 minutes)
5. Speak for 2–3 minutes using Gunn's Rule of 3: introduction, body, conclusion
6. Reflect on one strength and one goal

---

## Accessibility Features

- Read-aloud with adjustable voices
- High-contrast mode
- Variable spin speed
- Translation support (Spanish and French)

---

## The Architectural Approach

This tool was built using a faculty-centered, privacy-conscious workflow. The instructor defined the pedagogical logic — the prompts, the structure, the learning purpose. AI was used only for the mechanical assembly of the HTML scaffold. Student data never enters a proprietary system.

This approach is described in full in the accompanying conference paper presented at the International Communication Association (ICA) Annual Conference, 2026.
